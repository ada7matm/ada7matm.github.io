Centrado

(todos los trucos)


Horizontal

margin: auto


Si tenemos un elemento en bloque (display: block), y este tiene un ancho determinado (en px. o %), podemos centrarlo usando margin: 0 auto (el 0 no es obligatorio, puede ser una medida si queremos darle margen superior e inferior):

img {
 display: block;
 width: 200px;
 margin: 0 auto;
}




El elemento se centra siempre con respecto al ancho del elemento padre o contenedor:




Si tenemos varios elementos flotados, y estos elementos tienen ancho fijo, podemos encerrarlos en un div que los contenga, con ancho suficiente para que entren todos en línea, y centrar este div con la misma técnica. Poniendo overflow: hidden al contenedor, si todos sus hijos están flotados, evitamos que colapse.

img {
 display: block;
 width: 200px;
 margin: 10px;
 float: left;
}
.container {
 overflow: hidden;
 width: 660px;
 margin: 0 auto;
}





text-align: center

Si tenemos uno o varios elementos en línea, podemos aprovecharnos de su comportamiento y centrarlo usando text-align: center en el elemento que lo contiene


img {
 display: inline-block;
 width: 200px;
}
.container {
 text-align: center;
}






Hay que tener cuidado porque si algunos de estos elementos en línea, tienen texto, estos se alinearán verticalmente con línea de la base del texto por defecto

left: 50% y translateX

Si un elemento en bloque está posicionado, de cualquier forma que no sea estática, podemos usar left: 50% para moverlo a la mitad de su elemento contenedor, si está posicionado de forma relativa, o del elemento que tiene como referencia si está posicionado absolutamente. Como el left: 50% lo toma desde su punto (0,0), es decir, desde el extremo superior izquierdo, tenemos que moverlo hacia la izquierda la mitad de su ancho. Para esto usamos transform: translateX(-50%), para moverlo hacia atrás la mitad de su ancho

img {
 display: block;
 width: 200px;
 position: relative;
 left: 50%;
 transform: translateX(-50%);
}





flexbox


Con flexbox la tarea se vuelve mucho más sencilla. Simplemente necesitamos de un contenedor flex y de definirle justify-content: center


.container {
 display: flex;
 justify-content: center;
}

Vertical

line-height

Si tenemos texto en una sola línea, podemos utilizar la propiedad line-height para que el renglón tenga el mismo alto que el elemento que lo contiene.

.text {
 position: absolute;
 top: 0;
 line-height: 200px;
}

Gatito




padding-top

Si conocemos el alto del elemento y alto del contenedor, podemos tomar la diferencia y dividirla por dos, y asignarla al padding-top del contenedor

.element {
 height: 200px;
}
.container {
 height: 300px;
 padding-top: 50px;
 box-sizing: border-box;
}



top: 50% y translateY(-50%)

Si un elemento en bloque está posicionado, de cualquier forma que no sea estática, podemos usar top: 50% para moverlo a la mitad de su elemento contenedor, si está posicionado de forma relativa, o del elemento que tiene como referencia si está posicionado absolutamente. Como el top: 50% lo toma desde su punto (0,0), es decir, desde el extremo superior izquierdo, tenemos que moverlo hacia arriba la mitad de su alto. Para esto usamos transform: translateY(-50%), para moverlo hacia arriba la mitad de su alto

.text {
 position: absolute;
 top: 0;
 top: 50%;
 transform: translateY(-50%);
}

Horizontal y Vertical

text-align y line-height

Si es texto en una sola línea, y conocemos el alto del contenedor, podemos centrar el texto horizontalmente con text-align:center y verticalmente con la line-height equivalente al alto del contenedor

.text {
 line-height: 200px;
 text-align: center;
}

Texto centrado



margin y padding

Si es un elemento en bloque, y conocemos el alto del mismo y de su contenedor, podemos combinar las técnicas de margin: 0 auto para centrarlo horizontalmente, y darle al contenedor un padding-top correspondiente a la mitad de la diferencia de sus alturas

.element {
 height: 200px;
 margin: 0 auto;
 display: block;
}
.container {
 height: 300px;
 padding-top: 50px;
 box-sizing: border-box;
}



position y translate(-50%, -50%)

Si un elemento en bloque está posicionado, de cualquier forma que no sea estática, podemos combinar ambas técnicas de centrado, y usar top: 50%, left: 50% y transform: translate(-50%, -50%). El elemento padre debe tener un alto determinado para que funcione.

img {
 position: absolute;
 top: 50%;
 left: 50%;
 transform: translate(-50%, -50%);
}



flexbox

Con flexbox, nuevamente podemos centrar horizontal y verticalmente con facilidad. Si tenemos un contenedor flex con ancho y alto definido, podemos centrar sus elementos usando justify-content: center para centrarlo horizontalmente, y align-items: center para hacerlo verticalmente.

.container {
 display: flex;
 width: 400px;
 height: 300px;
 justify-content: center;
 align-items: center;
}
