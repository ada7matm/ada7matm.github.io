Pseudo clases


Los elementos html tienen estados que varían dependiendo de cómo interactuemos con ellos
Haciendo uso de las pseudo clases, podemos darle estilos particulares a dichos elementos según el
estado en que se encuentren.

La sintaxis para usar pseudo clases es:

selector:pseudo-clase {
 propiedad-a-modificar: valor;
}



Hover

Es la pseudo clase más utilizada. Es el estado al que cambia un elemento cuando el mouse está encima de este.
Nos sirve para realizar efectos que se activen cuando el cursor pasa por uno de ellos:

.example-one:hover {
 color: brown
 background-color: lightsalmon;
 cursor: pointer;
}

.example-two:hover {
 border-bottom: 3px solid black;
 cursor: pointer;
}

.example-three:hover {
 opacity: .8;
 cursor: pointer;
}

required


Se utiliza para dar estilos a los inputs que tienen el atributo required



.input:required {
 border-bottom: 4px solid goldenrod;
 border-left: 4px solid goldenrod;
}





valid

Se utiliza para dar estilos a los inputs cuando estos pasan al estado valid, es decir
cuando son completados correctamente según su tipo de datos


.input:valid {
 border-bottom: 4px solid green;
 border-left: 4px solid green;
}





invalid

Se utiliza para dar estilos a los inputs cuando estos pasan al estado invalid , es decir
cuando no están correctamente completados (o incompletos si son requeridos)

.input:invalid {
 border-bottom: 4px solid crimson;
 border-left: 4px solid crimson;
}





focus

Se utiliza para dar estilos a los inputs cuando estos tienen el foco, es decir,
cuando son selccionados

.input:focus {
 border-bottom: 4px solid darkblue;
 border-left: 4px solid darkblue;
}





nth-of-type(n)

Se utiliza para dar estilos a los elementos de un mismo tipo, en el mismo nivel jerárquico,
es decir, que sean hermanos, cuyo orden satisfaga la fórmula indicada en los paréntesis

Si sólo se indica un número n, se seleccionará el n elemento de dicho tipo, por ejemplo

p:nth-of-type(3) {
 color: red;
}

En cambio, si dentro de los paréntesis se utiliza una fórmula con el valor n, n irá tomando los valores sucesivos 1, 2, etc
y se irá aplicando a los elementos que estén en la posición resultante final

p:nth-of-type(2n) {
 color: red;
}

Esto se usa mucho a la hora de hacer grillas, ya que podemos definir
fácilmente que elemento va a pasar a una nueva línea

img {
 float: left;
}

img:nth-of-type(3n + 4) {
 clear: both;
}
